## 0.1.0

* Initial release of `composable_architecture_core`
* Store, Reducer, and Effect system with unidirectional data flow
* Lens and Prism optics for state composition
* Observable with `map`, `distinct`, and `when` operators
* Effect types: run, future, stream, cancel, debounce, throttle, merge, concatenate
* Reducer composition via `combine` and `pullback`
* `forEach` reducer for collection state management
